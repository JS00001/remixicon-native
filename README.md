# remixicon-native

RemixIcon for React Native

## Installation

```sh
npm install remixicon-native
```

## Usage

```js
import { RemixIcon } from 'remixicon-native';

// ...
<RemixIcon name="remixicon-line" color="black" size={24} />;
///...
```
